# 【从零手写播放器】第 17 章：字幕渲染

> 字幕是视频播放中的重要功能。本章我们来学习字幕的类型，使用 FFmpeg 解码字幕流，并通过 SDL_ttf 将字幕渲染到视频画面上。

## 17.1 字幕的类型

### 17.1.1 按存储方式分类

| 类型 | 说明 | 示例 |
| --- | --- | --- |
| **外挂字幕** | 独立的字幕文件，与视频文件分开 | .srt, .ass, .ssa, .vtt |
| **内嵌字幕** | 作为流封装在容器文件中 | MKV 中的字幕轨 |
| **硬字幕（烧录）** | 直接编码在视频画面中，无法关闭 | 某些在线视频 |

### 17.1.2 按格式分类

| 格式 | 类型 | 特点 |
| --- | --- | --- |
| **SRT** | 文本字幕 | 最简单，仅支持纯文本+时间戳 |
| **ASS/SSA** | 文本字幕 | 支持样式、位置、特效 |
| **VTT** | 文本字幕 | Web 标准（WebVTT） |
| **PGS** | 图形字幕 | 蓝光字幕，图片形式 |
| **DVB Sub** | 图形字幕 | 数字电视字幕 |

文本字幕需要渲染引擎（字体、排版），图形字幕直接叠加。

### 17.1.3 SRT 格式示例

```srt
1
00:00:01,000 --> 00:00:04,000
大家好，欢迎学习音视频开发教程。

2
00:00:05,000 --> 00:00:08,500
本章我们来学习字幕渲染。

3
00:00:09,000 --> 00:00:12,000
Let's get started!
```

格式非常简单：序号 → 时间范围 → 字幕文本。

## 17.2 FFmpeg 中的字幕处理

### 17.2.1 字幕流

字幕在 FFmpeg 中以 `AVMEDIA_TYPE_SUBTITLE` 类型的流存在：

```cpp
for (unsigned i = 0; i < fmt_ctx->nb_streams; i++) {
    if (fmt_ctx->streams[i]->codecpar->codec_type == AVMEDIA_TYPE_SUBTITLE) {
        std::cout << "字幕流 #" << i << ": "
                  << avcodec_get_name(fmt_ctx->streams[i]->codecpar->codec_id)
                  << std::endl;
    }
}
```

### 17.2.2 字幕解码

字幕解码与音视频不同，使用专用的 API：

```cpp
#include <libavcodec/avcodec.h>

AVSubtitle subtitle;
int got_sub = 0;

// 注意：字幕解码不使用 send/receive 模型
// 而是使用旧的 avcodec_decode_subtitle2 接口
int ret = avcodec_decode_subtitle2(subtitle_codec_ctx, &subtitle,
                                    &got_sub, pkt);

if (got_sub) {
    // subtitle.start_display_time: 显示开始时间（相对于包 PTS，毫秒）
    // subtitle.end_display_time:   显示结束时间（毫秒）
    // subtitle.num_rects:          矩形数量
    // subtitle.rects[]:            字幕矩形数组

    for (unsigned i = 0; i < subtitle.num_rects; i++) {
        AVSubtitleRect* rect = subtitle.rects[i];

        switch (rect->type) {
            case SUBTITLE_TEXT:
                // 纯文本字幕
                std::cout << "Text: " << rect->text << std::endl;
                break;

            case SUBTITLE_ASS:
                // ASS 格式字幕（含样式标记）
                std::cout << "ASS: " << rect->ass << std::endl;
                break;

            case SUBTITLE_BITMAP:
                // 图形字幕（位图）
                // rect->data[0]: RGBA 像素数据
                // rect->w, rect->h: 尺寸
                // rect->x, rect->y: 位置
                break;

            default:
                break;
        }
    }

    avsubtitle_free(&subtitle);
}
```

### 17.2.3 解析 ASS 字幕文本

ASS 格式的字幕文本通常包含样式信息，格式如下：

```
Dialogue: 0,0:00:01.00,0:00:04.00,Default,,0,0,0,,大家好，欢迎学习。
```

需要解析出纯文本：

```cpp
std::string extract_text_from_ass(const char* ass_line) {
    std::string text(ass_line);

    // 跳过前面的样式字段（以逗号分隔，取最后一个逗号后的内容）
    int comma_count = 0;
    size_t pos = 0;
    for (size_t i = 0; i < text.size(); i++) {
        if (text[i] == ',') {
            comma_count++;
            if (comma_count == 9) {  // ASS Dialogue 有 9 个逗号分隔的字段
                pos = i + 1;
                break;
            }
        }
    }

    std::string result = text.substr(pos);

    // 去除 ASS 样式标签 {\...}
    std::string clean;
    bool in_tag = false;
    for (char ch : result) {
        if (ch == '{') in_tag = true;
        else if (ch == '}') in_tag = false;
        else if (!in_tag) clean += ch;
    }

    // 将 \N 替换为换行
    size_t p;
    while ((p = clean.find("\\N")) != std::string::npos) {
        clean.replace(p, 2, "\n");
    }

    return clean;
}
```

## 17.3 使用 SDL_ttf 渲染文字

SDL_ttf 是 SDL2 的字体渲染扩展库。

### 17.3.1 安装

```bash
# Ubuntu
sudo apt install libsdl2-ttf-dev

# macOS
brew install sdl2_ttf
```

### 17.3.2 基本用法

```cpp
#include <SDL2/SDL_ttf.h>

// 初始化
TTF_Init();

// 打开字体
TTF_Font* font = TTF_OpenFont("/usr/share/fonts/truetype/dejavu/DejaVuSans.ttf", 24);
if (!font) {
    std::cerr << "无法打开字体: " << TTF_GetError() << std::endl;
}

// 渲染文字到 Surface
SDL_Color white = {255, 255, 255, 255};
SDL_Surface* surface = TTF_RenderUTF8_Blended(font, "Hello 你好", white);
```

> **中文字幕的字体选择**：上面示例使用的 `DejaVuSans.ttf` 是拉丁字体，**不包含中文字形**，中文字幕会显示为乱码或方块。显示中文字幕必须使用支持 CJK（中日韩）字符集的字体。
>
> 在 Ubuntu 上安装中文字体：
> ```bash
> sudo apt install fonts-noto-cjk
> ```
>
> 各平台常见的 CJK 字体路径：
>
> | 平台 | 字体路径 |
> | --- | --- |
> | Linux（Noto） | `/usr/share/fonts/opentype/noto/NotoSansCJK-Regular.ttc` |
> | Linux（文泉驿） | `/usr/share/fonts/truetype/wqy/wqy-zenhei.ttc` |
> | macOS | `/System/Library/Fonts/PingFang.ttc` |
> | Windows | `C:\Windows\Fonts\msyh.ttc`（微软雅黑） |
>
> 实际项目中应按优先级依次尝试多个字体路径，CJK 字体优先，拉丁字体兜底：
>
> ```cpp
> const char* font_paths[] = {
>     // CJK 字体优先
>     "/usr/share/fonts/opentype/noto/NotoSansCJK-Regular.ttc",
>     "/usr/share/fonts/truetype/wqy/wqy-zenhei.ttc",
>     "/System/Library/Fonts/PingFang.ttc",
>     "C:\\Windows\\Fonts\\msyh.ttc",
>     // 兜底（不支持中文，但至少能显示英文字幕）
>     "/usr/share/fonts/truetype/dejavu/DejaVuSans.ttf",
> };
>
> TTF_Font* font = nullptr;
> for (const auto* path : font_paths) {
>     font = TTF_OpenFont(path, 28);
>     if (font) break;
> }
> ```

```cpp
// 将 Surface 转为 Texture
SDL_Texture* text_texture = SDL_CreateTextureFromSurface(renderer, surface);

// 渲染到屏幕
SDL_Rect dst = {
    (win_w - surface->w) / 2,   // 居中
    win_h - surface->h - 50,    // 底部偏上
    surface->w,
    surface->h
};
SDL_RenderCopy(renderer, text_texture, nullptr, &dst);

// 释放
SDL_DestroyTexture(text_texture);
SDL_FreeSurface(surface);

// 程序结束时
TTF_CloseFont(font);
TTF_Quit();
```

### 17.3.3 字幕渲染函数

```cpp
void render_subtitle(SDL_Renderer* renderer, TTF_Font* font,
                      const std::string& text, int win_w, int win_h) {
    if (text.empty()) return;

    SDL_Color white = {255, 255, 255, 255};
    SDL_Color black = {0, 0, 0, 128};

    // 渲染阴影（黑色，偏移 2 像素）
    SDL_Surface* shadow_surface = TTF_RenderUTF8_Blended(font, text.c_str(), black);
    if (shadow_surface) {
        SDL_Texture* shadow_tex = SDL_CreateTextureFromSurface(renderer, shadow_surface);
        SDL_SetTextureAlphaMod(shadow_tex, 128);
        SDL_Rect shadow_rect = {
            (win_w - shadow_surface->w) / 2 + 2,
            win_h - shadow_surface->h - 48,
            shadow_surface->w,
            shadow_surface->h
        };
        SDL_RenderCopy(renderer, shadow_tex, nullptr, &shadow_rect);
        SDL_DestroyTexture(shadow_tex);
        SDL_FreeSurface(shadow_surface);
    }

    // 渲染正文（白色）
    SDL_Surface* text_surface = TTF_RenderUTF8_Blended(font, text.c_str(), white);
    if (text_surface) {
        SDL_Texture* text_tex = SDL_CreateTextureFromSurface(renderer, text_surface);
        SDL_Rect text_rect = {
            (win_w - text_surface->w) / 2,
            win_h - text_surface->h - 50,
            text_surface->w,
            text_surface->h
        };
        SDL_RenderCopy(renderer, text_tex, nullptr, &text_rect);
        SDL_DestroyTexture(text_tex);
        SDL_FreeSurface(text_surface);
    }
}
```

## 17.4 字幕同步

字幕的显示时机需要与音频时钟同步：

```cpp
struct SubtitleEntry {
    double start_time;   // 开始显示时间（秒）
    double end_time;     // 结束显示时间（秒）
    std::string text;    // 字幕文本
};

// 在渲染循环中
std::string current_subtitle;
for (const auto& sub : subtitle_entries) {
    double current_time = audio_clock.get();
    if (current_time >= sub.start_time && current_time <= sub.end_time) {
        current_subtitle = sub.text;
        break;
    }
}

if (!current_subtitle.empty()) {
    render_subtitle(renderer, font, current_subtitle, win_w, win_h);
}
```

## 17.5 Demo：简易外挂 SRT 字幕

由于完整的字幕集成涉及解码线程的扩展，这里展示一个简化的方案——加载外挂 SRT 文件：

```cpp
#include <fstream>
#include <sstream>
#include <vector>
#include <regex>

struct SubEntry {
    double start, end;
    std::string text;
};

// 解析时间 "HH:MM:SS,mmm" 为秒
double parse_srt_time(const std::string& s) {
    int h, m, sec, ms;
    sscanf(s.c_str(), "%d:%d:%d,%d", &h, &m, &sec, &ms);
    return h * 3600.0 + m * 60.0 + sec + ms / 1000.0;
}

// 加载 SRT 文件
std::vector<SubEntry> load_srt(const std::string& path) {
    std::vector<SubEntry> subs;
    std::ifstream file(path);
    if (!file.is_open()) return subs;

    std::string line;
    while (std::getline(file, line)) {
        // 跳过序号
        if (line.empty() || std::all_of(line.begin(), line.end(), ::isdigit))
            continue;

        // 解析时间行
        if (line.find("-->") != std::string::npos) {
            SubEntry entry;
            auto arrow = line.find("-->");
            entry.start = parse_srt_time(line.substr(0, arrow));
            entry.end = parse_srt_time(line.substr(arrow + 4));

            // 读取字幕文本（可能多行）
            std::string text;
            while (std::getline(file, line) && !line.empty()) {
                if (!text.empty()) text += " ";
                text += line;
            }
            entry.text = text;
            subs.push_back(entry);
        }
    }

    return subs;
}
```

在渲染循环中使用：

```cpp
// 加载字幕
auto subs = load_srt("subtitle.srt");

// 在渲染循环中
double cur_time = audio_clock.get();
std::string sub_text;
for (const auto& s : subs) {
    if (cur_time >= s.start && cur_time <= s.end) {
        sub_text = s.text;
        break;
    }
}

// 渲染视频帧后，叠加字幕
if (!sub_text.empty()) {
    render_subtitle(renderer, font, sub_text, win_w, win_h);
}
```
