## 18.2 核心知识点总结

### 18.2.1 播放器数据流

```
文件  →  [Demux]  →  AVPacket  →  [Decode]  →  AVFrame  →  [Render/Play]
               ↕              ↕               ↕
          PacketQueue      解码器缓冲      FrameQueue
```

### 18.2.2 关键 FFmpeg API

| 功能 | API |
| --- | --- |
| 打开文件 | `avformat_open_input()` |
| 获取流信息 | `avformat_find_stream_info()` |
| 查找最佳流 | `av_find_best_stream()` |
| 读取数据包 | `av_read_frame()` |
| 查找解码器 | `avcodec_find_decoder()` |
| 打开解码器 | `avcodec_open2()` |
| 送入数据包 | `avcodec_send_packet()` |
| 取出帧 | `avcodec_receive_frame()` |
| 图像格式转换 | `sws_scale()` |
| 音频重采样 | `swr_convert()` |
| Seek | `avformat_seek_file()` |
| 清空解码器 | `avcodec_flush_buffers()` |

### 18.2.3 音视频同步要点

1. 以音频时钟为主时钟
2. 音频时钟 = 最近一帧的 PTS + 已经过的时间
3. 视频帧根据 PTS 与音频时钟的差值调整等待时间
4. 视频落后太多时丢帧追赶

### 18.3.2 错误处理改进

```cpp
// 使用 C++ 异常或 std::expected（C++23）
class FFmpegError : public std::runtime_error {
    int error_code_;
public:
    FFmpegError(int code, const std::string& msg)
        : std::runtime_error(msg), error_code_(code) {}
};

// 使用 RAII 管理所有 FFmpeg 资源
class AVFrameGuard {
    AVFrame* frame_;
public:
    AVFrameGuard() : frame_(av_frame_alloc()) {}
    ~AVFrameGuard() { av_frame_free(&frame_); }
    AVFrame* get() { return frame_; }
    AVFrame* operator->() { return frame_; }
};
```

## 18.4 进阶方向

### 18.4.1 硬件加速解码

使用 GPU 进行视频解码，大幅降低 CPU 占用：

```
CPU 软解码：CPU 负担重，兼容性好
GPU 硬解码：CPU 几乎不参与，功耗低

常见硬件加速方案：
├── Linux: VAAPI (Intel/AMD), VDPAU (NVIDIA)
├── Windows: DXVA2, D3D11VA
├── macOS: VideoToolbox
├── NVIDIA: NVDEC / CUVID
└── 通用: Vulkan Video (新兴)
```

### 18.4.2 网络流播放

扩展播放器以支持网络视频流：

| 协议 | 用途 | 特点 |
| --- | --- | --- |
| RTMP | 直播推拉流 | 低延迟，Flash 时代遗产 |
| HLS | 点播/直播 | HTTP 分片，自适应码率 |
| DASH | 点播/直播 | HTTP 分片，更灵活 |
| RTSP/RTP | 实时流 | 监控、视频会议 |
| SRT | 直播推流 | 低延迟，抗丢包 |

FFmpeg 原生支持这些协议，只需要把文件路径换成 URL：

```cpp
// 播放网络流
avformat_open_input(&fmt_ctx, "rtmp://live.example.com/stream", nullptr, nullptr);
avformat_open_input(&fmt_ctx, "http://example.com/video.m3u8", nullptr, nullptr);
```

### 18.4.3 音视频录制与编码

播放器的"反向"操作——采集、编码、封装：

```
摄像头/麦克风
      │
      ▼
  采集原始数据（V4L2/ALSA/AVFoundation）
      │
      ▼
  编码（libx264/libx265/aac）
      │
      ▼
  封装（MP4/FLV/TS）
      │
      ▼
  写入文件 / 推流到服务器
```

### 18.4.4 音视频滤镜

FFmpeg 的 libavfilter 提供了强大的滤镜功能：

```
视频滤镜：
├── scale: 缩放
├── crop: 裁剪
├── overlay: 画中画
├── drawtext: 添加文字
├── rotate: 旋转
└── hflip/vflip: 翻转

音频滤镜：
├── volume: 音量调节
├── equalizer: 均衡器
├── aresample: 重采样
├── atempo: 变速不变调
└── amix: 混音
```

### 18.4.5 跨平台 GUI

将播放器集成到跨平台 GUI 框架中：

| 方案 | 特点 |

| Qt + FFmpeg | 最成熟的 C++ GUI 框架，QML 支持 |
