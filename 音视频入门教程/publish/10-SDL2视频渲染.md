# 【从零手写播放器】第 10 章：SDL2 视频渲染

> 前面我们把解码后的视频帧保存为图片文件。但播放器需要把画面**实时显示**在窗口中。本章我们引入 SDL2，学习如何将 FFmpeg 解码的视频帧渲染到屏幕上。

## 10.1 SDL2 简介

SDL2（Simple DirectMedia Layer 2）是一个跨平台的多媒体库，提供对以下系统功能的统一抽象：

- **视频**：窗口管理、2D/3D 渲染
- **音频**：音频播放
- **输入**：键盘、鼠标、游戏手柄
- **定时器**：高精度计时

SDL2 在底层使用各平台的原生 API：
- Linux：X11/Wayland + OpenGL/Vulkan
- Windows：Direct3D / GDI
- macOS：Cocoa + Metal/OpenGL

## 10.2 SDL2 核心概念

### 10.2.2 Window → Renderer → Texture

SDL2 的渲染架构是三层结构：

![SDL2 视频渲染](https://gitee.com/yuhong1234/ffmpeg-player-tutorial/raw/master/images/diagram-10-sdl2-rendering.png)

**Window**：一个操作系统窗口。

```cpp
SDL_Window* window = SDL_CreateWindow(
    "FFmpeg Player",              // 窗口标题
    SDL_WINDOWPOS_CENTERED,       // X 位置
    SDL_WINDOWPOS_CENTERED,       // Y 位置
    1280, 720,                    // 宽高
    SDL_WINDOW_SHOWN | SDL_WINDOW_RESIZABLE  // 标志
);
```

**Renderer**：关联到窗口的渲染器。

```cpp
SDL_Renderer* renderer = SDL_CreateRenderer(
    window, -1,
    SDL_RENDERER_ACCELERATED | SDL_RENDERER_PRESENTVSYNC
);
```

**Texture**：GPU 内存中的图像数据，可高效更新和渲染。

```cpp
SDL_Texture* texture = SDL_CreateTexture(
    renderer,
    SDL_PIXELFORMAT_IYUV,        // YUV420P 格式（I420）
    SDL_TEXTUREACCESS_STREAMING, // 可频繁更新
    width, height
);
```

### 10.2.3 关键：直接渲染 YUV，无需 CPU 格式转换

SDL2 的 Texture 原生支持 YUV 格式！我们可以直接将 FFmpeg 解码出的 YUV420P 数据送入 Texture，由 GPU 完成 YUV→RGB 的转换和渲染。

## 10.3 渲染流程

每帧的渲染流程：

```cpp
// 1. 将 YUV 数据更新到 Texture
SDL_UpdateYUVTexture(
    texture,
    nullptr,                   // 更新整个纹理
    frame->data[0], frame->linesize[0],  // Y 平面
    frame->data[1], frame->linesize[1],  // U 平面
    frame->data[2], frame->linesize[2]   // V 平面
);

// 2. 清空渲染器
SDL_RenderClear(renderer);

// 3. 将 Texture 拷贝到渲染器（可指定目标区域实现缩放）
SDL_RenderCopy(renderer, texture, nullptr, nullptr);

// 4. 呈现到屏幕
SDL_RenderPresent(renderer);
```

## 10.4 事件循环

SDL2 使用事件驱动模型。必须在主循环中处理事件，否则窗口会无响应：

```cpp
SDL_Event event;
bool running = true;

while (running) {
    // 处理所有待处理事件
    while (SDL_PollEvent(&event)) {
        switch (event.type) {
            case SDL_QUIT:           // 用户关闭窗口
                running = false;
                break;
            case SDL_KEYDOWN:
                if (event.key.keysym.sym == SDLK_ESCAPE) {
                    running = false;
                }
                break;
            case SDL_WINDOWEVENT:    // 窗口大小变化等
                break;
        }
    }

    // 解码和渲染
    // ...
}
```

## 10.5 Demo：FFmpeg 解码 + SDL2 实时渲染

这个 Demo 将 FFmpeg 解码的视频帧实时渲染到 SDL2 窗口中。注意，这个版本**不做帧率控制**，画面会以最快速度闪过。

```cpp
// chapter-10-sdl-video/main.cpp

extern "C" {
#include <libavformat/avformat.h>
#include <libavcodec/avcodec.h>
#include <libavutil/avutil.h>
}

#include <SDL2/SDL.h>
#include <iostream>
#include <iomanip>

int main(int argc, char* argv[]) {
    if (argc < 2) {
        std::cerr << "用法: " << argv[0] << " <输入文件>" << std::endl;
        return 1;
    }

    const char* input_file = argv[1];

    // ========== 1. FFmpeg: 打开文件 ==========
    AVFormatContext* fmt_ctx = nullptr;
    if (avformat_open_input(&fmt_ctx, input_file, nullptr, nullptr) < 0) {
        std::cerr << "无法打开文件" << std::endl;
        return 1;
    }
    avformat_find_stream_info(fmt_ctx, nullptr);

    int video_idx = av_find_best_stream(fmt_ctx, AVMEDIA_TYPE_VIDEO, -1, -1, nullptr, 0);
    if (video_idx < 0) {
        std::cerr << "找不到视频流" << std::endl;
        avformat_close_input(&fmt_ctx);
        return 1;
    }

    AVStream* video_stream = fmt_ctx->streams[video_idx];
    int width = video_stream->codecpar->width;
    int height = video_stream->codecpar->height;

    // ========== 2. FFmpeg: 初始化解码器 ==========
    const AVCodec* codec = avcodec_find_decoder(video_stream->codecpar->codec_id);
    AVCodecContext* codec_ctx = avcodec_alloc_context3(codec);
    avcodec_parameters_to_context(codec_ctx, video_stream->codecpar);
    avcodec_open2(codec_ctx, codec, nullptr);

    AVPacket* pkt = av_packet_alloc();
    AVFrame* frame = av_frame_alloc();

    // ========== 3. SDL2: 初始化 ==========
    if (SDL_Init(SDL_INIT_VIDEO) < 0) {
        std::cerr << "SDL 初始化失败: " << SDL_GetError() << std::endl;
        return 1;
    }

    SDL_Window* window = SDL_CreateWindow(
        "FFmpeg + SDL2 视频渲染",
        SDL_WINDOWPOS_CENTERED, SDL_WINDOWPOS_CENTERED,
        width, height,
        SDL_WINDOW_SHOWN | SDL_WINDOW_RESIZABLE
    );

    SDL_Renderer* renderer = SDL_CreateRenderer(
        window, -1,
        SDL_RENDERER_ACCELERATED | SDL_RENDERER_PRESENTVSYNC
    );

    // 创建 YUV 纹理
    SDL_Texture* texture = SDL_CreateTexture(
        renderer,
        SDL_PIXELFORMAT_IYUV,          // 对应 YUV420P
        SDL_TEXTUREACCESS_STREAMING,
        width, height
    );

    std::cout << "视频: " << width << "x" << height << std::endl;
    std::cout << "按 ESC 或关闭窗口退出" << std::endl;

    // ========== 4. 主循环：解码 + 渲染 ==========
    bool running = true;
    int frame_count = 0;
    Uint32 start_time = SDL_GetTicks();

    while (running) {
        // 处理事件
        SDL_Event event;
        while (SDL_PollEvent(&event)) {
            if (event.type == SDL_QUIT) {
                running = false;
            } else if (event.type == SDL_KEYDOWN) {
                if (event.key.keysym.sym == SDLK_ESCAPE) {
                    running = false;
                }
            }
        }

        if (!running) break;

        // 读取和解码
        int ret = av_read_frame(fmt_ctx, pkt);
        if (ret < 0) {
            // 文件结束，回到开头重新播放
            av_seek_frame(fmt_ctx, video_idx, 0, AVSEEK_FLAG_BACKWARD);
            avcodec_flush_buffers(codec_ctx);
            continue;
        }

        if (pkt->stream_index == video_idx) {
            avcodec_send_packet(codec_ctx, pkt);

            while (avcodec_receive_frame(codec_ctx, frame) == 0) {
                // 将 YUV 数据更新到 SDL Texture
                SDL_UpdateYUVTexture(
                    texture, nullptr,
                    frame->data[0], frame->linesize[0],
                    frame->data[1], frame->linesize[1],
                    frame->data[2], frame->linesize[2]
                );

                // 渲染
                SDL_RenderClear(renderer);
                SDL_RenderCopy(renderer, texture, nullptr, nullptr);
                SDL_RenderPresent(renderer);

                frame_count++;
                av_frame_unref(frame);
            }
        }

        av_packet_unref(pkt);
    }

    // ========== 5. 统计和清理 ==========
    Uint32 elapsed = SDL_GetTicks() - start_time;
    if (elapsed > 0) {
        double fps = frame_count * 1000.0 / elapsed;
        std::cout << "播放了 " << frame_count << " 帧, 耗时 "
                  << elapsed << "ms, 平均 " << std::fixed << std::setprecision(1)
                  << fps << " fps" << std::endl;
    }

    SDL_DestroyTexture(texture);
    SDL_DestroyRenderer(renderer);
    SDL_DestroyWindow(window);
    SDL_Quit();

    av_frame_free(&frame);
    av_packet_free(&pkt);
    avcodec_free_context(&codec_ctx);
    avformat_close_input(&fmt_ctx);

    return 0;
}
```

## 10.6 简单的帧率控制

虽然正式的帧率控制要等到音视频同步章节，但我们可以先加一个简单的延时：

```cpp
// 在渲染后添加延时
double fps = av_q2d(video_stream->avg_frame_rate);
double frame_delay = 1000.0 / fps;  // 每帧应该等待的毫秒数

// ...在渲染循环中:
SDL_RenderPresent(renderer);
SDL_Delay(static_cast<Uint32>(frame_delay));  // 简单延时
```

这种方式不精确（没有考虑解码和渲染的耗时），但能让画面以大致正确的速度播放。

## 10.7 窗口缩放

SDL2 的 `SDL_RenderCopy` 会自动将 Texture 缩放到目标区域。当窗口大小改变时，画面会自动拉伸填满窗口。

如果要**保持宽高比**，需要计算目标矩形：

```cpp
void calculate_display_rect(SDL_Rect* rect, int win_w, int win_h,
                             int video_w, int video_h) {
    float aspect = static_cast<float>(video_w) / video_h;
    int w = win_w;
    int h = static_cast<int>(w / aspect);

    if (h > win_h) {
        h = win_h;
        w = static_cast<int>(h * aspect);
    }

    rect->x = (win_w - w) / 2;
    rect->y = (win_h - h) / 2;
    rect->w = w;
    rect->h = h;
}

// 使用
SDL_Rect display_rect;
int win_w, win_h;
SDL_GetWindowSize(window, &win_w, &win_h);
calculate_display_rect(&display_rect, win_w, win_h, width, height);

SDL_RenderClear(renderer);
SDL_RenderCopy(renderer, texture, nullptr, &display_rect);
SDL_RenderPresent(renderer);
```
